# 🎮 Global Video Game Sales Analysis (Tableau)

An interactive Tableau dashboard **analysing** global video game sales between **1980 and 2010**. The dashboard explores sales trends, regional performance, platform popularity, publisher success, and genre performance through interactive visualisations.

## 📖 Project Overview

This project analyses historical video game sales data using Tableau. The dashboard presents the data in a clear and interactive way, helping users identify trends and gain business insights.

The dashboard allows users to:

- Explore global sales trends
- Compare regional sales
- Analyse platform and publisher performance
- Identify the best-selling games and genres

---

## 🎯 Business Questions

The dashboard answers questions such as:

- Which years had the highest video game sales?
- Which platforms were the most successful?
- Which publishers dominated the market?
- Which genres generated the highest sales?
- How did sales differ across regions?

---

## 📂 Dataset

The dataset contains historical video game sales between **1980 and 2010**, including:

- Game Name
- Platform
- Release Year
- Genre
- Publisher
- Regional Sales (North America, Europe, Japan and Other)
- Global Sales

**Dataset Source:** Kaggle – Video Game Sales Dataset

---

## 🧹 Data Preparation

Before creating the dashboard, the data was prepared by:

- Checking for missing values
- Reviewing data types
- Formatting the release year
- Validating regional and global sales
- Removing unnecessary fields

## 💡 Key Insights

- Nintendo was the leading publisher during the period analysed.
- North America recorded the highest overall video game sales.
- Action and Sports were the best-selling genres.
- Sales patterns varied across different regions.
