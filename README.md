# Global Video Game Sales Analysis (Tableau)

This project is an interactive Tableau dashboard built using historical video game sales data from 1980 to 2010. I created it to practise building dashboards in Tableau and to explore how sales changed across different platforms, publishers, genres, and regions over time.

The dashboard allows users to interact with the data using filters, making it easy to compare trends and explore different parts of the market.

---

## Project Objectives

- Prepare the video game sales data for analysis.
- Analyse historical sales trends using Tableau.
- Build an interactive dashboard with filters and dashboard actions.
- Compare sales across years, regions, platforms, publishers, and genres.
- Present key insights in a clear and interactive dashboard.

## Dataset

The dataset contains video game sales from **1980 to 2010**, with information about game titles, platforms, publishers, genres and regional sales.

To prepare the data for Tableau, I reviewed the dataset, checked for missing values, corrected data types where needed, and removed fields that weren't required for the dashboard.

| Column | Description |
|---------|-------------|
| `Name` | Video game title |
| `Platform` | Gaming platform |
| `Year` | Release year |
| `Genre` | Game genre |
| `Publisher` | Game publisher |
| `NA_Sales` | North America sales (millions) |
| `EU_Sales` | Europe sales (millions) |
| `JP_Sales` | Japan sales (millions) |
| `Other_Sales` | Other region sales (millions) |
| `Global_Sales` | Total global sales (millions) |

**Source:** Kaggle – Video Game Sales Dataset

##  Key Findings

Some of the main insights from the dashboard include:

- Nintendo was the highest-selling publisher during the period analysed.
- North America recorded the highest overall sales.
- Action and Sports were the most popular genres.
- Sales trends varied across different regions.
- Platform popularity changed significantly over the 30-year period.
